

# RecordLife 活着

一款用于记录生活、追踪人生数据的 HarmonyOS 移动应用。

## 应用简介

RecordLife（活着）是一款专为鸿蒙系统设计的个人生活记录应用，旨在帮助用户追踪和记录日常生活中的重要数据，包括时间线、日历事件、个人目标等。

## 主要功能

### 📅 日历视图
- 月历导航与浏览
- 按日期查看历史记录
- 快速跳转指定日期

### 📝 记录管理
- 添加生活记录
- 时间轴展示
- 标签分类管理
- 图片附件支持

### 👤 个人中心
- 用户资料编辑
- 出生日期设置
- 目标年龄规划
- 账户数据统计

### ⚙️ 设置与管理
- 应用主题切换（明暗模式）
- 数据备份与恢复
- 配置个性化选项

## 技术栈

- **框架**: HarmonyOS
- **语言**: ArkTS
- **最低版本**: API 9+

## 使用说明

1. **安装**: 将项目导入 DevEco Studio 后构建安装到真机或模拟器
2. **数据记录**: 在首页点击添加按钮记录生活点滴
3. **查看日历**: 通过日历视图浏览特定日期的记录
4. **账户管理**: 在账户页面查看统计数据和个人信息

## 版本要求

- HarmonyOS SDK: API 9+
- DevEco Studio: 3.0+

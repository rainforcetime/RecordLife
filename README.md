
# Record Life (记录)

一款基于 HarmonyOS 开发的生活记录应用，帮助用户记录生活中的重要时刻和日常点滴。

## 功能特点

- **时间轴记录**：按时间顺序浏览和添加生活记录
- **日历视图**：通过日历直观查看每日的记录情况
- **图片分享**：将记录以图片形式分享给朋友
- **数据备份**：支持本地数据备份和恢复

## 页面说明

| 页面 | 功能描述 |
|------|----------|
| 首页 | 时间轴展示今日记录 |
| 此刻 | 记录新的生活瞬间 |
| 我的 | 个人资料和设置中心 |

## 技术栈

- 框架：HarmonyOS ArkUI
- 开发语言：ETS (Extended TypeScript)
- 适配设备：华为手机/平板

## 安装使用

1. 确保已安装 DevEco Studio 开发环境
2. 使用 DevEco Studio 打开项目目录
3. 连接真机或使用模拟器运行调试

## 项目结构

